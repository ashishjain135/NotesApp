## Online Notes/Diary Application (MERN Stack)
This project is a secure online personal notes application built using the MERN stack (MongoDB, Express, React, Node.js).
Users can register and log in securely, create notes, edit them, delete them, and pin important notes at the top.
The app ensures that all the notes are stored safely in a database and can be accessed anytime from anywhere after login.
